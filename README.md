#  CityInclui

 ## Projeto universitário, criado para ajudar pessoas com deficiência visual.










## 🛠️ Stacks utilizadas 
**Front-end:**
- Next.js
- Funcionalidade de voz usando Web Speech API integrada à navegação por teclado

**Infraestrutura:**
 - Banco: PostgreSQL (Aurora/RDS) 
 - Armazenamento de imagens: Amazon S3
 - Deploy do backend: Elastic Beanstalk
 - Deploy do frontend: AWS Amplify 
 - Configuração de domínio: Route 53




##  Objetivo do projeto
O CityInclui é um site criado para ajudar pessoas com deficiência visual a encontrar restaurantes acessíveis, trazendo como diferencial uma funcionalidade de voz inteligente, capaz de fazer a leitura dinâmica da interface enquanto o usuário navega. Essa solução não é comum nem mesmo em sites grandes, pois a web costuma depender apenas de leitores de tela externos. Por isso, desenvolver um recurso próprio, integrado à navegação por teclado, foi um grande desafio técnico e um avanço importante em acessibilidade dentro do nosso projeto.
